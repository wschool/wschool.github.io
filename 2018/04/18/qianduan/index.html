网站建设所需要的前端知识 - 多年积累干货分享

By admin

发表于 2018-04-18

文章目录

学习前端基础知识：
1、HTML/HTML5：HTML 指超文本标签语言。
2、CSS/CSS3：指层叠样式表（Cascading Style Sheets）。
3、Javascript：脚本语言。

在这基础之上深造：
1、Jquery：兼容几乎所有浏览器的前端脚本库。
2、requireJS: 是一个非常小巧的JavaScript模块载入框架。
3、SeaJS: 是一个模块加载器,模块加载器需要实现两个基本功能: 实现模块定义规范,这是模块系统的基础。（建议用requireJS）
4、UI库：jQuery WeUI、Ant Design、layui、Flat UI，列举几个典型的，还有在后续文章中分享给大家。
5、字体图标：阿里巴巴矢量图标库、Font Awesome，还有其他字体图标，不一一列举了。
6、动画：animate.css、animejs，多年收集的动画库有几十个，后面文章分享给大家。

前端构建：
1、ES2015：是JavaScript语言的下一代标准。
2、Webpack：是什么webpack是一款模块加载器兼打包工具,它能把各种资源,例如JS(含JSX)、coffee、样式(含less/sass)、图片等都作为模块来使用和处理。
3、gulp：是一个自动化构建工具,开发者可以使用它在项目开发过程中自动执行常见任务。（建议用Webpack）

如果前端在往更强大的前端发展，基于MVVM驱动架构，构建单页面应用，多用于移动端开发，可选择以下一种：
1、Vuejs：（读音 /vjuː/, 类似于 view） 是一套构建用户界面的渐进式框架，只关注视图层， 采用自底向上增量开发的设计。
2、ReactJS： 是一个用于构建用户界面的 JAVASCRIPT 库，起源于 Facebook 的内部项目，用来架设 Instagram 的网站，并于2013年5月开源。
3、AngularJS：可以构建一个单一页面应用程序（SPAs：Single Page Applications）。

先简单了解前端知识，我会在后面文章中一一介绍。
